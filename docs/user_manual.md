# User and Operator Guide

Guide for HRDC Administrators, Trainers, and Participants using **LPU HRDC Nexus**.

---

## 1. HRDC Administrator Operations

### Creating programs and sessions
1. Log in with an `admin` or `staff` credential (e.g. `admin@lpu.co.in`).
2. Click **Create Programme** in the top bar.
3. Fill out the title, category (FDP, Workshop, Orientation), mode (online, offline, hybrid), start/end dates, and capacity limit.
4. Once created, click on the programme card to add sessions and schedule timelines.

### Auditing logs and billing
- Navigate to the **Corporate Portal** inside the sidebar.
- Track client contract documents, verify client employee attendance counts, and check payment/invoice histories.
- Use the **AI Assistant** to query attendance averages, find students with low attendance levels, and generate FDP summaries.

---

## 2. Trainer Operations

### Managing materials and tests
1. Enter your Trainer Portal (`trainer@lpu.co.in`).
2. Select your active course session in the list.
3. Click **Upload Syllabus Notes**. Select a PDF file, PowerPoint deck, or text notes.
4. The backend will parse the file text, split it into paragraphs, compute vector embeddings, and inject the content into the AI RAG Knowledge Base.
5. Create MCQ quizzes or coding assignments. Use the submission tabs to evaluate subjective essay questions.

---

## 3. Participant Operations

### Registering and attending
1. Access the web app from desktop or smartphone (PWA install popup).
2. Register as a **Participant Faculty** or log in.
3. Select any active programme from the grid directory and click **Enroll**.
4. To clock in for a class session:
   - Navigate to the **Attendance Hub**.
   - Input the rolling session ID code.
   - Click **Scan & Verify Check-In**.
   - The app reads your smartphone coordinates to confirm you are within LPU campus geofence limits.
5. Review lessons under course syllabus and claim your digital certificates when attendance is above 75%.
